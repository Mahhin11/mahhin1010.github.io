Sales Funnel Performance Dashboard

This interactive dashboard was created as part of a freelance project in July 2024. It offers real-time visualization of sales funnel performance with drill-through capabilities and KPI tracking.

Project Details

  • Connected directly to Excel through SharePoint to create near real-time updating dashboard, eliminating manual refresh processes
  • Implemented 30+ complex DAX queries and 15+ bookmark states behind 5 button clicks which previously required entering data into 8 different fields to get same view
  • Created interactive Data Visualization with drill-through capabilities, custom tooltips, and KPI metrics for performance tracking
  • Saved client team 8 hours of weekly manual reporting work by upgrading their Excel dashboard into self-reporting Power BI dashboard
